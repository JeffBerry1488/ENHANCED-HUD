Display Enhancer

Replaces the default PolyTrack speedometer with a detailed HUD overlay featuring real-time speed, G-force meter, drift angle, and lateral grip indicator.

Data is read directly from the physics engine for maximum accuracy. No settings needed — works automatically.

Press T to toggle the overlay on/off.

Features

  • Large digital speed readout (km/h)
  • Animated arc gauge with color shift
  • Canvas G-force dot indicator
  • Drift angle display
  • Lateral grip readout
